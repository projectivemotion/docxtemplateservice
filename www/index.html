Free Docx Document Generator Service

Generate a word document using custom values.

How to use

Simply send a post request (multipart/form-data) to

Curl Example

curl -o parsed.docx --form docxfile=@template.docx --form jsondata='{ "title": "hello from curl!" }' http://localhost:3000/

Curl Example 2

curl -o parsed.docx --form docxfile=@template.docx --form jsondata=@example.json http://localhost:3000/

Php example

                $post = array('docxfile' => '@/path/to/template.docx','jsondata'=> '@/path/to/values.json');
                $ch = curl_init();
                curl_setopt($ch, CURLOPT_URL,$target_url);
                curl_setopt($ch, CURLOPT_POST,1);
                curl_setopt($ch, CURLOPT_POSTFIELDS, $post);
                $result=curl_exec ($ch);
                curl_close ($ch);
